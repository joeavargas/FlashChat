# FlashChat
Chat App using Googe's Firestore for user authentication and databasing

# Libraries used
* Google Firebase/Firestore
* IQKeyboardManagerSwift via Swift Package Manager(SPM)

# Description
Using Google's Firestore make it easy to authenticate users securely and deliver messages in blink-of-an-eye real-time fast!
